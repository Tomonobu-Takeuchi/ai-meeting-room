プライバシーポリシー

AI-Paradise（AI-Persona会議室・Digital-Persona相談室）

基本方針

竹内智信（以下「当社」といいます）は、AI-Paradise（AI-Persona会議室・Digital-Persona相談室。以下「本サービス」といいます）の運営にあたり、ユーザーの個人情報を適切に保護することを重要な責務と考えています。本プライバシーポリシーは、当社が収集する情報の種類、利用目的、および管理方法を説明するものです。

本サービスをご利用いただくことにより、本プライバシーポリシーに同意したものとみなします。

第1条（収集する情報）

当社は、以下の情報を収集することがあります。

アカウント情報
  • メールアドレス（登録・ログイン・お問い合わせ対応に使用）
  • パスワード（bcryptによりハッシュ化して保存。平文では保存しません）
  • 表示名（任意）
利用情報
  • ペルソナデータ（名前・説明・性格設定等、ユーザーが入力した情報）
  • 学習データ（テキスト・URL・音声・動画・PDFなど、ユーザーが提供したデータ）
  • 会議・会話の記録（会議ログ・PDF議事録）
  • 利用プラン・会議回数・クレジット残数
決済情報
  • 決済処理はStripe, Inc.が行います。クレジットカード番号等の決済情報は当社のサーバーには保存されません。
  • 当社が保持するのは、StripeのカスタマーID・決済セッションID・プラン情報のみです。
自動収集情報
  • アクセスログ（IPアドレス・ブラウザ情報・アクセス日時）※サーバーログとして一定期間保持
  • セッション情報（Flaskセッション。ログイン状態の維持に使用）

収集しない情報：当社は、氏名・住所・電話番号・マイナンバー等の個人識別情報をサービス利用に際して収集しません。会話・学習データへのこれら情報の入力はお控えください。

第2条（情報の利用目的）

収集した情報は、以下の目的にのみ利用します。

情報の種類利用目的
メールアドレスログイン認証・重要なお知らせの送付
ペルソナ・学習データAIペルソナの生成・会議機能の提供
会議ログPDF議事録の生成・過去会議の参照
利用状況データプラン管理・利用制限の適用・サービス改善
アクセスログセキュリティ監視・障害対応

上記以外の目的での利用は行いません。利用目的を変更する場合は、ユーザーへ事前に通知します。

第3条（外部サービスへのデータ送信）

本サービスは、機能提供のために以下の外部サービスにデータを送信します。各サービスのプライバシーポリシーも合わせてご確認ください。

サービス運営会社・所在地送信データ用途
Claude API Anthropic, PBC（米国） 会話テキスト・ペルソナ設定 AI会話の生成
OpenAI API OpenAI, Inc.（米国） テキスト・音声データ 文字起こし（Whisper）・Embedding生成
ElevenLabs ElevenLabs, Inc.（米国） テキスト・音声サンプル 音声生成（Premiumプラン）
HeyGen HeyGen Inc.（米国） 写真・テキスト アバター動画生成（Phase2以降）
Stripe Stripe, Inc.（米国） 決済情報 課金処理

音声データまたは画像・動画データを外部APIに送信する際には、送信前に同意確認画面を表示します。同意しない場合、当該機能はご利用いただけません。

第4条（データの保管・管理）

  1. ユーザーデータはRailway（米国）が提供するPostgreSQLデータベースに保存されます。
  2. パスワードはbcryptにより不可逆なハッシュ値として保存し、平文では保存しません。
  3. 秘密モードでのセッションデータは暗号化して保存し、デフォルトで30日後に自動削除します。
  4. 当社は、不正アクセス・データ漏洩・改ざんを防止するための適切なセキュリティ対策を講じます。ただし、インターネット上での完全な安全性を保証することはできません。
  5. 当社はユーザーデータを第三者（上記外部APIを除く）に販売・提供しません。

第5条（データの保持期間）

データの種類保持期間
アカウント情報アカウント削除まで
ペルソナ・学習データアカウント削除まで（または手動削除）
会議ログアカウント削除まで（または手動削除）
秘密モードデータデフォルト30日（設定変更可）
アクセスログ最大90日
決済履歴法令に基づき7年間保持

第6条（ユーザーの権利）

ユーザーは、自身のデータについて以下の権利を行使できます。

📋 開示請求
当社が保持するご自身のデータの開示を請求できます。
✏️ 訂正・削除
不正確なデータの訂正、または特定データの削除を依頼できます。
🗑️ アカウント削除
アカウント削除時は、関連するすべてのペルソナ・学習データも削除されます。
📤 データエクスポート
サービス終了時は、ペルソナ・学習データのエクスポート機能を提供するよう努めます。

権利行使のご依頼は、お問い合わせフォームよりご連絡ください。合理的な期間内に対応します。

第7条（Cookieについて）

  1. 本サービスは、ログイン状態の維持のためにFlaskセッションクッキーを使用します。このクッキーは本サービスの基本機能に必要であり、無効化するとログイン機能が利用できません。
  2. 本サービスは、広告目的のトラッキングクッキーや第三者広告クッキーを使用しません。
  3. Stripe Inc.が決済処理のためにクッキーを使用する場合があります。詳細はStripeのプライバシーポリシーをご確認ください。

第8条（未成年者について）

  1. 本サービスは、14歳未満の方を対象としていません。
  2. 18歳未満の方がご利用の場合は、保護者の同意を得た上でご利用ください。
  3. 14歳未満の方が誤って登録した場合、保護者からの申し出によりデータを削除します。

第9条（プライバシーポリシーの変更）

  1. 当社は、必要に応じて本プライバシーポリシーを変更することができます。
  2. 重要な変更がある場合は、サービス上での掲示またはメールにより事前にユーザーへ通知します。
  3. 変更後のプライバシーポリシーは、本サービス上に掲示した時点から効力を生じます。

第10条（お問い合わせ）

個人情報の取り扱いに関するご質問・ご要望・苦情は、本サービス上のお問い合わせフォームよりご連絡ください。遅滞なく対応いたします。
